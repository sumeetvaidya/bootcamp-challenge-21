# Bootcamp Challenge 21 - Monetary System for Mars Kasei Coin
A monetary system for the new Mars colony. A new monetary system on blockchain technology, and to define a new cryptocurrency called KaseiCoin. (“Kasei” means “Mars” in Japanese.)
## Technologies 
<br/>
The project is written in Solidity
<br/>


## Installation Guide  
<br/>
The program is compiled and deployed and run in Remix. 
<br/>
